🚗 Civic — auto-fires on CarPlay disconnect
  1. Shortcuts app → Automation → + → CarPlay → Disconnects → Run Immediately
  2. Add action: Find Items in Find My → filter Name = "Car"
  3. Add action: Get Details of Find My Items → choose Location
  4. Add action: Get Details of Locations → choose Latitude (repeat for Longitude)
  5. Add action: Format Date → current date → custom format X (Unix timestamp)
  6. Add action: URL → paste your Civic URL, replace LAT/LNG/TS with the magic variables
  7. Add action: Open URLs
🏍️ G310 R — NFC tag or Lock Screen widget
NFC (helmet tap):
  1. Shortcuts → Automation → + → NFC → scan your helmet tag → Run Immediately, no notification
  2. Same actions as above but Find My item = "BMW G310R" and source=nfc
Lock Screen widget (fallback):
  1. Create a manual shortcut "Log Bike Parked" with the same actions
  2. Settings → Wallpaper → Customize Lock Screen → Widgets → Shortcuts → pick it

Note: AirTag location updates every few minutes. If you just parked, the coordinates should be fresh — but if Find My is slow, the app will show a manual picker as fallback.
